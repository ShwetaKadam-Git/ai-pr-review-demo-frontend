# ai-pr-review-demo-frontend

Dashboard UI for browsing review history from [ai-pr-review-demo-api](https://github.com/ShwetaKadam-Git/ai-pr-review-demo-api).

Part of the [ai-pr-review-platform](https://github.com/ShwetaKadam-Git/ai-pr-review-platform) ecosystem.

## Run locally

```bash
npm install
cp .env.example .env   # set VITE_API_BASE_URL
npm run dev
```

## Deployment

Deployed to GitHub Pages on every push to `main`. Zero cost.
